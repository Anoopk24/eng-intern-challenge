# Go Instructions
Translate your braille to text or text to braille!!

Sample command to translate to braille:
```
go run translator.go Abc 123 xYz
```

```
go run translator.go .....OO.....O.O...OO...........O.OOOO.....O.O...OO..........OO..OO.....OOO.OOOO..OOO
